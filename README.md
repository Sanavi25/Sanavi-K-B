# Responsive Landing Page

##  Project Description
This project is a simple responsive landing page created using HTML and CSS as part of a Web Development Internship Task.

## 🚀 Features
- Header with logo and navigation menu
- Hero section with heading, paragraph, and call-to-action button
- Footer with social media links
- Fully responsive design using media queries

## 🛠 Technologies Used
- HTML5
- CSS3
- Flexbox
- Media Queries

## 📱 Responsiveness
The website is responsive and works on:
- Mobile devices
- Tablets
- Desktop screens

## 📂 Project Structure
- index.html → Main webpage structure
- style.css → Styling and layout
- README.md → Project documentation

## 🎯 Outcome
This project helped me understand:
- HTML structure
- CSS layout techniques
- Responsive web design
